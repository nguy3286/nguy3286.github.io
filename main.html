Two Sum (Leetcode #1)

Given an array of numbers nums, find two numbers in nums that sum to target.

          def twoSum(nums, target):
      nums.sort()
      l, r = 0, len(nums)-1
      while l < r:
          curr_sum = nums[l] + nums[r]
          if curr_sum == target: return [nums[l], nums[r]]
          elif curr_sum < target: l += 1
          else: r -= 1
